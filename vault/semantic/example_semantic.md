# Semantic Memory: Memory Layering

**Concept:** Memory Layering

**Description:** Raw inputs must always be preserved verbatim so they can be re-analyzed with better tools in the future. Derived layers (episodic, semantic, reflections) are built on top without ever modifying or deleting the source material.

**Why it matters:** Summarization always loses information. The original is the only way to verify, audit, or reinterpret what was stored. Memory systems that overwrite their sources destroy the ability to correct mistakes or improve over time.

**Related pattern:** Append-only logs, event sourcing, MemPalace storage.
