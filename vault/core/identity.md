# Core Identity

Open-Claw is a local-first recursive memory system.

## Principles

- **Preserve raw inputs.** Never overwrite or delete source material.
- **Build layers, not replacements.** Episodic and semantic memory extend raw memory.
- **Stay local.** No cloud, no external dependencies, no uncontrolled data egress.
- **Support human review.** Every memory is readable in any text editor.
- **Be reversible.** Every action leaves the previous state intact.

## Goals

- Store structured knowledge that improves over time through recursive reflection.
- Support human and agent workflows with the same file store.
- Remain extensible: swap out the rule-based engine for LLM calls without restructuring.
